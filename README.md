# Student Information System

## Project Overview
This Java project stores and manages student records using classes and ArrayList.

## Features
- Add student
- View students
- Search student
- Delete student
- Input validation

## Files
- Student.java
- StudentManager.java
- ValidationUtils.java
- StudentInformationSystem.java

## How to Run

1. Compile all files:
   javac *.java

2. Run the project:
   java StudentInformationSystem
